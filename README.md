# Decision-Tree
Дерево решений — это алгоритм машинного обучения, который последовательно разделяет данные на группы на основе значений признаков. Он перебирает значения признаков и выбирает разделение по тому признаку, который даст максимальный прирост информации
Алгоритм дерева работы дерева решений:
1. Поиск лучшего вопроса для разбиения данных
2. Разделение объектов на двве группы по этому вопросу
3. Повтороение 1 и 2 пунктов для каждой подвыборки
4. Остановка когда достигнут критерий остановки

### Как не переобучить дерево решений?
Так как этот алгоритм всегда переобучается, задача программиста свести это переобучение к миниммуму, используя различные методы. В моем случае - критерии останова: Максимальная глубина дерева, Минимальное количество объектов в одном листе, Максимальное количество листьев.
